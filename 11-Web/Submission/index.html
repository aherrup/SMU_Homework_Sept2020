Analysis of the Weather and Latitude

below is a summary of a data analysis completed comparing different weather metrics based on a cities latitude.

Please click the button below to view the data we used in completing this analysis

View Data

This website is showing an analysis completed in November 2020 on different weather metrics taken from a variety of cities. In the project, we used OpenWeather API to collect data from over 500 cities accross the world with varing distances from the equator.

Once we collected the data we used Python to analyze what we had collected. We were looking to see if there was correlation between the latitude of a city and four key weather metrics: the city's tempurature, wind speed, cloudiness and humidity. We used MatPlotLib to create the visualizations (shown below) comparing these metrics.

This site will provide you with the four visualizations that were created comparing the varying weather metrics, the data we used to create these visualizations, along with descriptions and explinations for the trends we observed while completing this project.
